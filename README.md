# HireMetry

UI prototype for an AI interview practice product.

## Web UI

Location: `web/`

### Run locally (macOS / zsh)

```zsh
cd "$(pwd)/web"
npm install
npm run dev
```

Open the URL Vite prints (usually `http://localhost:5173`).

### Production build

```zsh
cd "$(pwd)/web"
npm run build
npm run preview
```
